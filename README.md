# Usage

To clone and use this template type the following commands:

```sh
git clone https://github.com/jayveedev-alt/my-app.git
```

```sh
cd my-app
```

Then, install dependencies:

## npm

```sh
npm install
```

```sh
npm run dev
```

<br />

> [!TIP]
> Remember to update the project name inside the `package.json` file.
